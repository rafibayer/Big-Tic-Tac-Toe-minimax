# Big-Tic-Tac-Toe-minimax
NxN Tic Tac Toe AI using Minimax with Alpha-beta pruning

# Features
- Timed AI via iterative deepening
- Max search-depth
- Memoized state and subtree evaluations
- Watch AI play or play against them yourself!
